Step1: git clone https://github.com/himon9/CRA-Vetting-List-Automation.git
Step2: Create virutal environment 'venv', and activate the virtual environment
Step3: Install dependencies: selenium, openpyxl in the venv
Step4: Enter your empId in Line 43 of the code ("C:/Users/<your-emp-id>/AppData/Local/Google/Chrome/User Data")
Step5: Run main.py using 'python main.py'
Step6: The automation script will take the client names from the file 'client_names.txt', fetch the results
        from OneView and store the results into client_data.xlsx file.
Step7: Relax and enjoy a cold beer, let the script do its work!
